# 🏦 Loan Approval Prediction

## 📌 Project Overview

This project predicts whether a loan application is likely to be approved based on applicant information such as income, education, employment status, credit history, loan amount, and other financial attributes. 
---

## 🎯 Objectives

- Clean and preprocess the dataset
- Perform Exploratory Data Analysis (EDA)
- Handle missing values and encode categorical variables
- Evaluate model performance
- Predict loan approval status
